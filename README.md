# CodeAlpha_Basic-chatbot

#**Basic Chatbot**#


***Description***

This project was developed as part of the CodeAlpha Python Programming Internship.

The Basic Chatbot is a simple rule-based conversational program that interacts with users through predefined responses. It accepts user input and replies based on specific keywords or phrases such as greetings, questions, and exit commands. This project demonstrates the fundamentals of chatbot development using Python.


***Features***

Interactive console-based chatbot

Responds to predefined user inputs

Simple and easy-to-use interface

Continuous conversation using loops

Exit option using a specific command

Beginner-friendly Python project

***Technologies Used***
